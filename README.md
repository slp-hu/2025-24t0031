# Music Recommendation Integrating Emotion and Genre via Multi-label Ranking Learning

## Overview
This study proposes a music recommendation system that integrates emotion and genre information.  
Conventional music recommendation is usually presented as a ranked list, making it difficult for users to understand how similar or different the next song will sound before listening.

To address this problem, this work constructs a commercial-music dataset with both emotion tags and genre labels, and trains a multimodal model on it.  
The results are then visualized in an embedding space, allowing users to explore music more intuitively by observing perceptual differences among songs.
